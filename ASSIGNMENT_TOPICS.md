# Python 程序设计大作业固定题目

请选择下列 10 个题目之一完成。不要自由另选完全无关题目；如确需调整，需提前获得老师确认。

每个题目都必须同时提供 CLI 和 Streamlit 页面。

## 1. 学习任务与时间管理系统

记录学习任务、课程、时长、截止日期、完成状态，支持统计学习时间和任务完成率。

## 2. 个人记账分析系统

记录收入和支出，按日期、分类、金额统计，支持搜索和月度汇总。

## 3. 运动健康记录系统

记录运动项目、时长、强度、日期、备注，展示运动总量和趋势摘要。

## 4. 读书 / 观影收藏系统

记录作品名称、类型、评分、状态、标签和短评，支持搜索、筛选和排行榜。

## 5. 成绩分析与学习建议系统

录入课程成绩，计算平均分、最高分、等级分布，并给出学习建议。

## 6. 习惯打卡看板

记录每日习惯打卡，统计连续天数、完成率和最近记录。

## 7. 校园活动报名统计系统

管理活动、报名人、签到状态和人数统计，适合班级或社团场景。

## 8. 个人任务看板

管理 Todo / Doing / Done 任务、优先级、截止日期，支持状态流转和统计。

## 9. 课程笔记检索系统

管理文本笔记，支持标签、关键词搜索、正则或模糊匹配、导入导出。

## 10. AI 学习复盘记录系统

记录学习问题、AI 回答摘要、自己的理解、复盘标签，支持搜索和复盘统计。
